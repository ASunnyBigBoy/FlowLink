# 流联(FlowLink)

一个基于Python的局域网文件传输和屏幕共享工具，支持在电脑和手机之间快速传输文件、查看手机屏幕和远程控制电脑。

## 功能特性

### 文件传输器 (FilePasser)
- Web界面管理，支持拖拽上传
- 多文件同时传输
- 实时文件列表和下载管理
- 手机扫码快速访问
- 支持文件夹创建和删除操作

### 手机屏幕显示器 (ShowScreen)
- 实时显示手机屏幕
- 可调节分辨率和帧率
- 低延迟传输
- 支持USB调试模式连接

### 电脑屏幕查看器 (ShowPC)
- 实时查看电脑屏幕
- 手机端触控控制电脑鼠标
- 支持截图和实时流模式
- 画质可调节

## 系统要求

- Python 3.7+
- Windows/macOS/Linux
- 手机和电脑需在同一WiFi网络

## 安装步骤

1. 克隆或下载项目文件
2. 安装依赖：
```bash
pip install -r requirements.txt
```

3. 运行主程序：
```bash
python main.py
```

## 使用方法

1. 运行程序后选择相应功能：
   - 输入 `1` 启动手机屏幕显示
   - 输入 `2` 启动文件传输服务  
   - 输入 `3` 启动电脑屏幕共享

2. 根据提示在手机浏览器访问显示的地址或扫描二维码

3. 开始使用相应功能

## 文件传输功能

- 电脑端：直接拖拽文件到上传区域或点击选择文件
- 手机端：访问Web界面后点击上传按钮
- 文件保存在桌面 `FilePasser` 文件夹中

## 屏幕共享功能

### 手机屏幕显示
- 需开启手机USB调试模式
- 连接手机到电脑
- 程序自动检测并显示手机屏幕

### 电脑屏幕共享
- 启动服务后手机访问相应地址
- 点击屏幕可远程控制电脑鼠标
- 支持全屏查看和实时刷新

## 注意事项

- 确保设备在同一局域网内
- 防火墙可能阻止连接，请允许程序访问网络
- 文件传输服务会占用8000-8100端口
- 使用后及时关闭服务以确保安全
